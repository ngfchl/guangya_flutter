## 1. 安装与登录

### 下载安装

- **macOS**：从 Release 页面下载 `.dmg` 安装包，拖入 Applications 即可。
- **Windows**：下载 `.exe` 安装包运行。
- **iOS / Android**：通过 TestFlight（iOS）或直接安装 APK。

### 登录

打开应用后会显示登录页面，支持以下登录方式：

- **扫码登录**（推荐）：在手机端小黄鸭 App 扫描二维码，自动完成登录。
- **账号密码**：输入账号密码登录（需服务端支持）。

登录成功后自动进入主界面。

## 2. 主界面概览

主界面分为左右两栏：

| 区域 | 说明 |
|------|------|
| **左侧面板** | 云盘目录树 / 媒体库切换 / 搜索入口 |
| **右侧面板** | 文件列表 / 影视内容 / 详情视图 |

顶部导航栏提供两个模式切换：

- 📁 **光鸭云盘**（Cloud Workspace）— 文件管理
- 🎬 **光鸭影视**（Media Center）— 影视媒体库

## 3. 光鸭云盘

### 3.1 文件浏览

- 支持**双面板**浏览，可通过切换按钮打开/关闭第二面板。
- 支持**跨面板拖拽**移动文件。
- 支持从系统 **Finder / 文件管理器**直接拖入上传。
- 支持下拉刷新。

### 3.2 文件操作

长按或右键点击文件可执行：

| 操作 | 说明 |
|------|------|
| 重命名 | 修改文件/文件夹名称 |
| 移动 | 移动到其他目录或另一个面板 |
| 复制 | 复制到目标目录 |
| 删除 | 删除文件（可多选） |
| 下载 | 下载到本地 |
| 分享 | 生成分享链接 |
| 详情 | 查看文件属性（大小、路径、修改时间等） |
| 打开 | 在系统默认程序中打开文件 |

### 3.3 云下载与离线任务

- 支持添加磁力链接、种子文件进行离线下载。
- 支持 HTTP / HTTPS 链接直接下载。
- 任务列表实时显示下载进度、速度和状态。

### 3.4 分享管理

- 创建分享链接并设置有效期。
- 管理已有分享（查看、取消）。
- 分享链接支持密码保护。

### 3.5 批量重命名

在「文件工具」中可批量重命名文件：

- 支持正则替换、序号递增等模式。
- 支持预览重命名结果后再执行。
- 支持撤销操作。

## 4. 光鸭影视

### 4.1 媒体库管理

点击「光鸭影视」进入影视模式，再点击 **媒体库管理** 按钮（齿轮图标）打开管理面板。

#### 新建媒体库

1. 点击「+ 新建」。
2. 输入媒体库名称（如「电影」「电视剧」）。
3. 选择类型：电影 / 电视剧 / 混合。
4. 选择云盘中的根目录作为来源。
5. 保存后自动开始扫描识别。

#### 管理已有媒体库

- **编辑**：修改名称、类型、来源目录。
- **删除**：删除媒体库及其所有刮削数据。

### 4.2 刮削识别

媒体库创建后会自动扫描文件并从 TMDB / 豆瓣获取影视信息：

- **自动识别**：根据文件名自动匹配影视作品。
- **批量识别**：对所有未匹配文件重新识别。
- **单个识别**：右键单个文件选择「手动匹配」。

识别结果包含：
- 中英文标题
- 海报 / 背景图
- 年份、类型、评分
- 剧情简介
- 是否含中文音轨 / 字幕
- 所属合集（系列）

### 4.3 手动匹配

当自动识别不准确时，可手动匹配：

1. 在文件或作品上选择「手动匹配」。
2. 在弹出的匹配对话框中：
   - **标题搜索**：输入片名，点击「搜索」按钮。
   - **ID 直达**：直接输入 TMDB ID 或豆瓣 ID，点击对应按钮直达详情。
3. 选择类型（电影 / 电视剧）和年份辅助筛选。
4. 从结果列表中选择正确的作品。
5. 确认后应用到该作品的所有资源版本。

### 4.4 详情页

点击影视作品可查看完整详情：

- 海报和背景图（可点击放大）
- 中英文标题、年份、类型标签
- 评分（TMDB / 豆瓣）
- 剧情简介
- 资源列表（所有关联文件）
- 每个资源可单独操作（播放、重命名、移动、删除）

### 4.5 视频播放

支持直接在应用内播放视频：

- 基于 **media_kit** 播放引擎，支持硬解和软解。
- 支持 macOS / Windows / Android / iOS。
- 支持字幕切换、音轨选择。
- 播放器支持全屏、进度拖拽、倍速播放。
- 播放退出时弹出确认，防止误操作。

### 4.6 备份与恢复

在媒体库管理面板中提供数据备份功能：

| 操作 | 说明 |
|------|------|
| 导出数据 | 将本地刮削数据库导出为 `.sqlite3` 文件 |
| 导入数据 | 从本地文件导入刮削数据库 |
| 同步到云盘 | 将刮削数据备份到云盘「小黄鸭备份」目录 |
| 从云盘恢复 | 从云盘下载备份并覆盖本地数据库 |

恢复完成后自动重新加载整个影视页面。

## 7. 常见问题

### Q：刮削识别失败怎么办？

1. 检查文件名是否包含可识别的关键词（片名、年份）。
2. 尝试手动匹配（输入片名搜索或直接输入 TMDB / 豆瓣 ID）。
3. 检查 TMDB API Key 是否配置正确。
4. 检查网络代理设置（TMDB 在部分地区需要代理）。

### Q：备份恢复后数据不见了？

恢复操作会**覆盖**本地刮削数据库。恢复完成后页面会自动重新加载。如果数据仍然异常，尝试重新扫描媒体库。

### Q：视频无法播放？

1. 检查文件格式是否支持（mp4、mkv、avi 等常见格式）。
2. macOS 上可能需要授予「辅助功能」权限。
3. 尝试在设置中切换硬解/软解模式。

### Q：Android 安装包被标记为风险？

这是安全软件误报，因为应用声明了文件管理相关权限。应用本身不包含任何恶意代码，可以安全使用。

### Q：macOS 上应用图标显示问号？

这是 macOS LaunchServices 索引问题，与应用无关。尝试重启 Mac 或执行：

```bash
sudo mdutil -E /
```

重建 Spotlight 索引后重启即可恢复。
