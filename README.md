## 1. 安装与登录

### 下载安装

- **macOS**：下载 `.dmg` 安装包，拖入 Applications 即可。
- **Windows**：下载 `.exe` 安装包运行。
- **iOS / Android**：通过 TestFlight（iOS）或直接安装 APK。

### 登录

打开应用后显示登录页面：

- **扫码登录**（推荐）：在手机端小黄鸭 App 扫描二维码自动完成登录。
- **账号密码**：输入账号密码登录。

## 2. 主界面概览

### 左侧菜单栏

应用左侧是菜单栏，包含两个核心入口和各种工具：

| 区域 | 说明 |
|------|------|
| **光鸭云盘** | 点击进入文件管理模式，浏览和管理云盘文件 |
| **光鸭影视** | 点击进入影视媒体库模式，管理刮削数据、播放视频 |

菜单栏底部还有设置入口。移动端点击左上角汉堡菜单可展开完整菜单。

### 右侧面板

根据当前模式显示对应内容：

- **云盘模式**：文件目录树、文件列表、文件预览
- **影视模式**：媒体库首页、作品详情页、视频播放器

### 模式切换

在菜单栏中点击「光鸭云盘」或「光鸭影视」即可切换模式，两个模式共享同一个应用但数据和功能完全独立。

## 3. 光鸭云盘

进入云盘模式后，左侧菜单栏会显示云盘相关的文件操作入口。

### 文件浏览

- 支持**双面板**浏览，可通过工具栏按钮开启/关闭第二面板。
- 支持**跨面板拖拽**移动文件。
- 支持从系统 **Finder / 文件管理器**直接拖入上传。
- 支持下拉刷新。
- 左侧有「最近查看」「照片」「视频」「音频」「文档」「分享」「回收站」等快捷分类入口。

### 文件操作

长按或右键点击文件：

| 操作 | 说明 |
|------|------|
| 重命名 | 修改文件/文件夹名称 |
| 移动 | 移动到其他目录或另一个面板 |
| 复制 | 复制到目标目录 |
| 删除 | 删除文件（可多选批量删除） |
| 下载 | 下载到本地 |
| 分享 | 生成分享链接 |
| 详情 | 查看文件属性（大小、路径、修改时间等） |
| 打开 | 在系统默认程序中打开文件 |

### 云下载与离线任务

- 支持添加磁力链接、种子文件进行离线下载。
- 支持 HTTP / HTTPS 链接直接下载。
- 任务列表实时显示下载进度、速度和状态。

### 分享管理

- 创建分享链接并设置有效期。
- 管理已有分享（查看、取消）。
- 分享链接支持密码保护。

## 4. 光鸭影视

点击左侧菜单栏的「光鸭影视」进入影视模式。

### 4.1 媒体库管理

进入影视模式后，点击工具栏的 **齿轮图标**（或菜单栏中的「媒体库管理」）打开管理面板。

#### 新建媒体库

1. 点击「+ 新建」。
2. 输入媒体库名称（如「电影」「电视剧」）。
3. 选择类型：电影 / 电视剧 / 混合。
4. 选择云盘中的根目录作为来源。
5. 保存后自动开始扫描识别。

#### 管理已有媒体库

- **编辑**：修改名称、类型、来源目录。
- **删除**：删除媒体库及其所有刮削数据。
- **备份**：导出/导入刮削数据库，同步到云盘或从云盘恢复。

### 4.2 刮削识别

媒体库创建后自动扫描文件并从 TMDB / 豆瓣获取影视信息：

- **自动识别**：根据文件名自动匹配影视作品。
- **批量识别**：对所有未匹配文件重新识别。
- **单个识别**：右键单个文件选择「识别」。

识别结果包含：中英文标题、海报/背景图、年份、类型、评分、剧情简介、是否含中文音轨/字幕、所属合集。

### 4.3 手动匹配

当自动识别不准确时：

1. 在作品或文件上选择「手动匹配」。
2. 输入片名点击「搜索」，或输入 TMDB / 豆瓣 ID 点击对应按钮直达。
3. 可选填类型（电影/电视剧）和年份辅助筛选。
4. 从结果列表中选择正确的作品。
5. 确认后应用到该作品的所有资源版本。

### 4.4 详情页

点击影视作品查看完整详情：

- 海报和背景图（可点击放大）
- 中英文标题、年份、类型标签
- 评分（TMDB / 豆瓣）
- 剧情简介
- 资源列表（所有关联文件）
- 每个资源可单独操作（播放、重命名、移动、删除）

### 4.5 视频播放

- 基于 **media_kit** 播放引擎，支持硬解和软解。
- 支持 macOS / Windows / Android / iOS。
- 支持字幕切换、音轨选择、全屏、进度拖拽、倍速播放。
- 播放退出时弹出确认，防止误操作。

### 4.6 备份与恢复

在媒体库管理面板的齿轮菜单中：

| 操作 | 说明 |
|------|------|
| 导出数据 | 将本地刮削数据库导出为 `.sqlite3` 文件 |
| 导入数据 | 从本地文件导入刮削数据库 |
| 同步到云盘 | 将刮削数据备份到云盘「小黄鸭备份」目录 |
| 从云盘恢复 | 从云盘下载备份并覆盖本地数据库 |

## 5. 工具箱

点击左侧菜单栏中的工具入口打开对应工具。云盘模式下显示「文件工具」，影视模式下显示「影视工具」。

### 5.1 文件扫描与清理

**入口**：菜单栏 → 文件工具 → 文件扫描与清理

扫描当前目录下的文件，检测并清理：

| 功能 | 说明 |
|------|------|
| **空文件夹** | 扫描并列出没有任何内容的文件夹，可一键批量删除 |
| **重复文件** | 通过文件大小 + MD5 校验识别完全相同的文件，支持按组查看并清理 |
| **相似文件夹** | 忽略空格、标点、副本后缀后名称相同的文件夹，检测误创建的重复目录 |

操作流程：
1. 选择要扫描的云盘目录。
2. 点击「开始扫描」，等待扫描完成。
3. 查看扫描结果，勾选要清理的项目。
4. 点击「删除选中」执行清理。

### 5.2 批量重命名

**入口**：菜单栏 → 文件工具 → 批量重命名

对云盘中的文件进行批量重命名：

1. **选择目标目录**：选择要重命名的文件所在的文件夹。
2. **设置过滤条件**：可按原文件名关键词过滤，只处理匹配的文件。
3. **添加重命名规则**：支持多种规则类型：
   - **查找替换**：将文件名中的指定文本替换为新文本。
   - **正则替换**：使用正则表达式进行高级匹配和替换。
   - **添加前缀/后缀**：在文件名前后添加文本。
   - **序号递增**：按顺序为文件添加数字编号。
4. **预览结果**：查看每条规则应用后的效果，确认无误后应用。
5. **执行重命名**：批量应用所有选中的重命名变更。

### 5.3 秒传工具

**入口**：菜单栏 → 文件工具 → 秒传工具

利用云盘秒传机制，将本地文件信息快速上传到云盘，无需实际传输文件数据。

操作方式：

- **粘贴 JSON**：粘贴秒传 JSON 数据（从其他客户端或分享获得）。
- **选择文件**：从本地选择文件生成秒传信息。
- **生成秒传 JSON**：将云盘文件导出为秒传 JSON，供其他客户端使用。

流程：
1. 选择目标云盘目录。
2. 输入或生成秒传 JSON 数据。
3. 确认文件列表后提交任务。
4. 任务列表显示上传进度和状态。

### 5.4 文件整理

**入口**：菜单栏 → 影视工具 → 文件整理

将媒体库中的文件按照规则自动整理到规范目录：

- 按影视作品名称创建文件夹。
- 将散落的媒体文件移动到对应的作品文件夹。
- 支持预览整理结果后再执行。

### 5.5 分类管理

**入口**：菜单栏 → 影视工具 → 分类管理

为影视媒体库配置自动分类规则：

- 按 TMDB 原始语言匹配分类。
- 支持自定义分类名称和匹配条件。
- 未匹配到任何规则的资源会归入默认分类。
- 支持为电影和剧集分别配置分类规则。

## 7. 常见问题

### Q：刮削识别失败怎么办？

1. 检查文件名是否包含可识别的关键词（片名、年份）。
2. 尝试手动匹配（输入片名搜索或直接输入 TMDB / 豆瓣 ID）。
3. 检查 TMDB API Key 是否配置正确。
4. 检查网络代理设置（TMDB 在部分地区需要代理）。

### Q：备份恢复后数据不见了？

恢复操作会**覆盖**本地刮削数据库。恢复完成后页面会自动重新加载。如果数据仍然异常，尝试重新扫描媒体库。

### Q：视频无法播放？

1. 检查文件格式是否支持（mp4、mkv、avi 等常见格式）。
2. macOS 上可能需要授予「辅助功能」权限。
3. 尝试在设置中切换硬解/软解模式。

### Q：Android 安装包被标记为风险？

这是安全软件误报，因为应用声明了文件管理相关权限。应用本身不包含任何恶意代码，可以安全使用。

### Q：macOS 上应用图标显示问号？

这是 macOS LaunchServices 索引问题，与应用无关。尝试重启 Mac 或执行：

```bash
sudo mdutil -E /
```

重建 Spotlight 索引后重启即可恢复。
